# WEBGuard API Documentation

## Overview

WEBGuard is a phishing detection platform that combines:

- Machine Learning URL Analysis
- Deep Learning URL Classification
- Computer Vision Screenshot Analysis
- Browser Extension Integration

The backend is built using FastAPI and exposes REST APIs used by the browser extension and dashboard.

```
http://localhost:8000
```

# API Endpoints

## 1. URL Analysis

### Endpoint

```http
POST /scan-url
```

### Description

Analyzes a website URL using the WEBGuard detection pipeline.

The pipeline extracts URL-based security features and performs classification using machine learning models.

### Request Body

```json
{
    "url": "https://example.com"
}
```

### Response

```json
{
    "prediction": "safe",
    "confidence": 73.4
}
```

### Possible Predictions

| Value | Meaning |
|---------|---------|
| safe | Website appears legitimate |
| suspicious | Potential phishing indicators detected |
| malicious | Strong phishing indicators detected |

---

## 2. Visual Screenshot Analysis

### Endpoint

```http
POST /analyze-visual
```

### Description

Analyzes a screenshot of the currently visited website.

The visual pipeline performs screenshot inspection using computer vision techniques to identify suspicious webpage characteristics.

### Request Body

```json
{
    "image": "<base64_image>"
}
```

### Response

```json
{
    "status": "success",
    "prediction": "safe",
    "confidence": 10,
    "saved_image": "debug_images/scan_xxxxx.png"
}
```

### Response Fields

| Field | Description |
|---------|---------|
| status | Request status |
| prediction | Visual classification |
| confidence | Risk score generated by CV engine |
| saved_image | Debug screenshot path |

# Browser Extension Integration

The Chrome Extension performs:

1. Automatic URL monitoring
2. Real-time URL scanning
3. Screenshot capture
4. Threat notifications
5. Popup dashboard updates
6. Badge status updates

# Threat Levels

## SAFE

Website shows minimal phishing indicators.

---

## SUSPICIOUS

Website exhibits unusual characteristics that may indicate phishing behavior.

---

## MALICIOUS

Website strongly matches phishing or malicious patterns and should be avoided.
